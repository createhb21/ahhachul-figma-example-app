# 아하철 × MUSINSA Example App

무신사 스타일 시안을 Vite + React 앱으로 패키징한 예제 프로젝트입니다.

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 구조

- `src/MusinsaStyleApp.jsx`: 메인 단일 파일 앱 컴포넌트
- `src/App.jsx`: Vite 엔트리용 얇은 래퍼
- `src/index.css`: 앱 구동에 필요한 최소 글로벌 스타일
- `tasks/todo.md`: 이 저장소 이식 작업 체크리스트와 검토 메모
- `tasks/lessons.md`: 이후 유사 작업에 재사용할 레슨
